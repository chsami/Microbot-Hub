# Humidifier Plugin

The **Humidifier Plugin** provides automation for humidifying items.

---

## Feature Overview

| Feature                 | Description                                   |
|-------------------------|-----------------------------------------------|
| **Items**               | Choose the Item to Humidify                   |

---

## Requirements
- Microbot RuneLite client
- Plugin enabled in the Microbot plugin list
- Required Levels, Items and Runes for humidifying

---

## How It Works
1. Configure the Settings to your liking.
2. Start the plugin near a bank.
3. Enjoy! :)
