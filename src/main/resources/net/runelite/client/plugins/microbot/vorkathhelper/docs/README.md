# Vorkath Helper Plugin

The **Vorkath Helper Plugin** displays useful overlays and helps in fighting Vorkath.

---

## Feature Overview

| Feature                      | Description                                                                                                       |
|------------------------------|-------------------------------------------------------------------------------------------------------------------|
| **Fireball Danger Zone**     | Displays fireball blast radius and ticks until it lands                                                           |
| **Acid tiles**               | Marks acid tiles during acid phase before they spawn in                                                           |
| **Vorkath Special Counter**  | Has a display counter showing progress until next special of Vorkath and indicates which special Vorkath will use |
| **Zombie Spawn Overlay**     | Marks zombie spawn with text and a line drawn from player so its easily visible                                   |
| **Auto Cast Crumble Undead** | Automatically casts crumble undead on the spawn if player has required runes (no Slayer staff required)           |

---

## Requirements
- Runes to cast crumble undead if Auto Cast Crumble Undead is turned on

---

## How It Works
Setup overlay colors in options
