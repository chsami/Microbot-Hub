# Farming Contracts

![icon.png](assets/icon.png)

Automates Guildmaster Jane's farming contracts in the Farming Guild. Start the plugin while inside the guild and it runs the full loop: take a contract, gather supplies, plant, harvest, and turn in.

## Features

| Feature | Description |
|---|---|
| **Contract handling** | Takes a contract from Jane and turns it in when complete. |
| **Auto banking** | Withdraws the tools, seeds, compost, and coins needed for the current contract. |
| **All patch types** | Plants and harvests allotment, herb, flower, bush, cactus, tree, and fruit tree contracts. |
| **Tree clearing** | Pays the guild gardeners (Rosie, Nikkie) to clear old or check-healthed trees before replanting. |
| **Weed handling** | Rakes patches before planting and re-rakes if weeds regrow during composting. |
| **Optional composting** | Fills the compost bin with pineapples for ultracompost. |
| **Tree protection** | Optionally pays a gardener to protect newly planted trees and fruit trees. |
| **Auto downgrade** | Requests an easier contract when the assigned crop's seeds are unavailable. |

## Requirements

- Inside the Farming Guild with the matching farming level for the contract tier.
- Seeds, compost, and coins available in the bank.
- Standard farming tools (spade, rake, seed dibber, secateurs).
